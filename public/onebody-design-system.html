Currently on the website — monochrome on light bg, no "Church" wordmark

⬤ ONEBODY
← no gradient, no "Church"
💡
The "Church" wordmark (italic script) must appear below the icon on all primary placements (website header, footer, OG images). The icon-only variant is only for platforms where the audience is already familiar — e.g. YouTube avatar.

Logo Rules

✓ Do

  • Use gradient version on light backgrounds
  • Use white version on dark photos / dark sections
  • Maintain clear space equal to the logo height on all sides
  • Include "Church" wordmark in primary usage

✗ Don't

  • Stretch, rotate, skew, or distort the logo
  • Use on busy / multicolour backgrounds
  • Use monochrome version on white / off-white backgrounds
  • Add words or extra design elements alongside the logo
